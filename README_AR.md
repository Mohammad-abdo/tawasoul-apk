## 🔄 العودة للباك اند

عند الرغبة في الربط مع الباك اند:

1. شغل الباك اند:
```bash
cd backend
npm run dev
```

2. عدّل `mahara_service.dart`:
```dart
// استبدل
import '../../shared/mock_content.dart';
// بـ
import '../../../core/services/api_service.dart';
```

3. باقي الشاشات تعمل تلقائياً مع الباك اند!
