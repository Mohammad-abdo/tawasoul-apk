## 🎨 اختبار الميزات

### 1. الفلترة والبحث
- افتح شاشة المنتجات
- اضغط على أيقونة الفلتر
- جرب الفلترة حسب:
  - الفئة
  - السعر
  - التقييم
  - التوفر

### 2. إضافة للسلة
- افتح أي منتج
- حدد الكمية
- اضغط "أضف إلى السلة"
- انتقل للسلة لعرض المنتجات

### 3. حجز موعد
- تصفح قائمة الأطباء
- اختر طبيب
- حدد موعد متاح
- أكمل الحجز

### 4. أنشطة مهارة
- افتح قسم الأنشطة
- جرب أنواع مختلفة:
  - استمع واختر
  - طابق الصور
  - رتب العناصر

## 🔄 الانتقال للباك اند

### عند الرغبة في الربط:

1. **تشغيل الباك اند:**
```bash
cd backend
npm run dev
```

2. **تحديث mahara_service:**
```dart
// استبدل في lib/features/mahara/services/mahara_service.dart
import '../../../core/services/api_service.dart';
```

3. **باقي الشاشات:**
- تعمل تلقائياً مع الباك اند
- Fallback للبيانات الثابتة عند فشل الاتصال

## ✨ نصائح للعمل الفعّال

### 1. تنظيم الكود
```
features/
├── shared/
│   └── mock_content.dart  ← البيانات هنا
├── products/
├── appointments/
└── ...
```

### 2. التعديل على البيانات
- افتح `mock_content.dart`
- ابحث عن القسم المطلوب
- عدّل أو أضف بيانات
- احفظ وأعد تشغيل التطبيق

### 3. إضافة شاشة جديدة
1. أنشئ ملف الشاشة
2. أضف البيانات في `mock_content.dart`
3. استخدم `MockContent.yourData` في الشاشة
4. أضف الـ route في `app_router.dart`
