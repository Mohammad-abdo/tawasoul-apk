## 🎮 جرب الميزات

### 1. تسجيل الدخول
1. افتح التطبيق
2. أدخل أي بريد وكلمة مرور
3. اضغط "تسجيل الدخول"
4. ستدخل مباشرة! ✨

### 2. تصفح المواعيد
1. من الشاشة الرئيسية
2. اذهب إلى "حجوزاتي"
3. ستجد مواعيد تجريبية

### 3. حجز موعد جديد
1. اذهب إلى "الأطباء"
2. اختر طبيب
3. اختر موعد
4. أكمل الحجز
5. سيظهر في قائمة حجوزاتك!

### 4. تصفح المنتجات
1. اذهب إلى "المتجر"
2. استخدم الفلتر
3. اضغط على منتج
4. أضفه للسلة

## 💡 نصائح مهمة

### التسجيل عبر OTP:
1. أدخل أي رقم هاتف (11 رقم على الأقل)
2. املأ البيانات
3. سيتم "إرسال" OTP (محاكاة)
4. أدخل أي 4 أرقام (مثال: 1234)
5. ستدخل مباشرة!

### البيانات:
- جميع البيانات محلية
- التغييرات لا تُحفظ بين الجلسات
- عند إعادة تشغيل التطبيق، البيانات ترجع للحالة الأولية

### الاختبار:
- جرب جميع الميزات
- لا تقلق من الأخطاء
- البيانات وهمية ومأمونة

## 🐛 حل المشاكل

### المشكلة: "Not authenticated"
**الحل:** سجل دخول أولاً

### المشكلة: البيانات لا تظهر
**الحل:** 
1. تأكد من تسجيل الدخول
2. انتقل للشاشة مرة أخرى
3. انتظر قليلاً (محاكاة تأخير الشبكة)

### المشكلة: التطبيق لا يعمل
**الحل:**
```bash
flutter clean
flutter pub get
flutter run
```

### المشكلة: أخطاء في التجميع
**الحل:**
```bash
flutter doctor
flutter pub upgrade
```

## 🎨 التخصيص

### إضافة بيانات جديدة:
1. افتح `lib/features/shared/mock_content.dart`
2. أضف بياناتك
3. احفظ
4. أعد تشغيل التطبيق

### مثال - إضافة طبيب:
```dart
{
  'id': 'doc_5',
  'name': 'د/ أحمد محمد',
  'specialty': 'تخاطب',
  'rating': 5.0,
  'reviewsCount': 200,
  // ... باقي البيانات
}
```
