# Mahara Kids Activity Screens

شاشات أنشطة مطابقة لتصميم تطبيق "Mahara Kids" للأطفال من 2-6 سنوات.

## الشاشات المتوفرة

### 1. Listen & Watch (استمع وشاهد)
- صورة كبيرة في المنتصف
- تشغيل صوت تلقائي عند فتح الشاشة
- زر إعادة تشغيل الصوت كبير أسفل الصورة
- أنيميشن تشجيعي بعد انتهاء الصوت

### 2. Listen & Choose (استمع واختر)
- صوت تلقائي في الأعلى
- 2-3 صور قابلة للاختيار في المنتصف
- أنيميشن اهتزاز عند الاختيار الخاطئ
- أنيميشن توهج عند الاختيار الصحيح

### 3. Matching (المطابقة)
- صور على اليسار
- أيقونات صوتية على اليمين
- سحب أو نقر للمطابقة
- المطابقة الصحيحة تُقفل مع أنيميشن

### 4. Sequence (التسلسل)
- بطاقات صور قابلة للسحب
- مناطق إسقاط واضحة
- الترتيب الخاطئ يعيد نفسه
- الترتيب الصحيح يظهر احتفال

### 5. Audio Association (ربط صوتي)
- صورة كبيرة سياقية
- صوت تلقائي عند التحميل
- زر إعادة تشغيل فقط

## الاستخدام

```dart
// من خلال Router
context.push('/mahara-kids/activity/activity_1?type=listen_watch');

// أو مباشرة
MaharaActivity activity = MaharaActivity(
  id: 'activity_1',
  type: 'listen_watch',
  audioUrl: 'https://...',
  mainImageUrl: 'https://...',
);

Widget screen = MaharaActivityRouter.buildScreen(activity);
```

## أنواع الأنشطة

- `listen_watch` - استمع وشاهد
- `listen_choose` - استمع واختر
- `matching` - المطابقة
- `sequence` - التسلسل
- `audio_association` - ربط صوتي

- ✅ عناصر كبيرة وسهلة
